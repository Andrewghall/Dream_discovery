# DREAM Output Demo - Complete Populated Template

## Goal
Create a fully populated, elegant workshop output template that demonstrates all 6 tabs with realistic content using collapsible accordions (not verbose walls of text).

## Tabs to Build:

### 1. Executive Summary ✅
- Solution overview with uploaded image
- High-level summary text (collapsible sections)
- Key metrics/highlights
- Workshop snapshot

### 2. Discovery Output 🔨
- Strategic tables by domain (Product, Technology, Commercial, Clinical)
- Each domain in collapsible accordion
- Design Principles (collapsible)
- Linked Intelligence sidebar

### 3. Reimagine ✅
- Three Houses Framework (already compact with accordions)
- Future vision opportunities (collapsible by domain)

### 4. Constraints 🔨
- Regulatory constraints (collapsible)
- Technical constraints (collapsible)
- Commercial constraints (collapsible)
- Organizational constraints (collapsible)

### 5. Commercial 🔨
- Investment summary
- Delivery phases (collapsible timeline)
- ROI projections
- Risk assessment

### 6. Summary 🔨
- Key findings (collapsible)
- Recommended next steps (collapsible)
- Success metrics
- Timeline overview

## Design Principles:
1. **Collapsed by default** - Show only headers
2. **Expand on click** - Reveal details with smooth animation
3. **Visual hierarchy** - Icons, colors, clear sections
4. **Scannable** - Can grasp content without expanding everything
5. **Professional** - Like PAM Wellness output, not "puking words"
